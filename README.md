# МедТрекер

Курсовой проект по дисциплине «Разработка веб-приложений»

**Тема:** Разработка веб-приложения для учёта и контроля приёма лекарств

## Описание

Веб-приложение реализует двухролевую модель взаимодействия «врач — пациент».

**Пациент** ведёт дневник приёма лекарств: отмечает принятые дозы, прикрепляет фотографии рецептов, самостоятельно добавляет курсы лечения и просматривает статистику соблюдения терапии.

**Врач** назначает курсы лечения пациентам, отслеживает статистику соблюдения назначений, просматривает историю приёмов за выбранный период и оставляет комментарии к курсу.

## Функциональность

- Двухролевая аутентификация (врач / пациент)

- Назначение курсов лечения врачом и самостоятельное добавление пациентом

- Дневник приёмов с недельным календарём и цветовой индикацией статусов

- Автоматический расчёт процента соблюдения курса

- Предупреждение при пересечении времени приёма разных препаратов

- Прикрепление фотографий рецептов

- Экспорт истории приёмов в PDF

## Технологии

- Python 3.13, Flask 3.0

- SQLite 3, Flask-SQLAlchemy

- Jinja2, Bootstrap 5

- ReportLab (генерация PDF)

- Flask-Login (аутентификация)
